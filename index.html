About project

Problem Statement

IITG BSc DSAI students need a local-first system to calculate official evaluation outcomes, track GPA and CGPA, explore syllabus data, plan targets, and export reports without a backend or private data upload.

Solution Architecture

Static UI Local Storage Evaluation Engine Chart.js Analytics PWA Cache GitHub Pages

Technology Stack

HTML5 CSS3 Vanilla JavaScript Chart.js CDN jsPDF CDN html2canvas CDN

Challenges Solved

  • Official trimester-specific IITG best-of evaluation rules.
  • Zero-backend persistence through Local Storage.
  • Offline-first deployment with service-worker caching.
  • Curriculum, GPA, syllabus, report, and analytics modules in one static app.

Future Enhancements

Relative GradingPeer percentile simulations when anonymized cohort data is available.
Study PlannerCalendar-free weekly plans stored locally.
